Detailed Description

Class that stores multistep activity that can be performed on multiple actuators.

There are two typed of steps that an action can consist of : WaitStep (delay between two actions) and StepStepActuator (action that is performed on a specific Actuator)

Parameters
m_activityStepsvector of steps that belongs to the activity.
m_actuatorslist of all the actuators that are in the action
