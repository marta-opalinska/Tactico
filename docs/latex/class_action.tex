\doxysubsection{Detailed Description}
Class that stores multistep activity that can be performed on multiple actuators. 

There are two typed of steps that an action can consist of \+: \mbox{\hyperlink{class_wait_step}{Wait\+Step}} (delay between two actions) and Step\+Step\+Actuator (action that is performed on a specific Actuator)
