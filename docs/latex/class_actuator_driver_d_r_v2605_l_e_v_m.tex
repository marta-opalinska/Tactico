\doxysubsection{Detailed Description}
Class that stores configuration and methods for DRV2605L evalution board actuator\textquotesingle{}s driver (controlled through I2C) 

The driver can have up to 7 waveforms (or wait times) preconfigured
\begin{DoxyItemize}
\item each need to be assigned to a specific slot. The slots will be played when \char`\"{}go\char`\"{} pin is triggered 
\end{DoxyItemize}
